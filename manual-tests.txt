# Module 6: UI Redesign - Neutral Monochromatic Theme

## Test Steps
1. Build and run CiCo using `./build.sh` and `./run.sh` (or use an IDE).
2. Observe the UI layout and colors across all screens.

## Expected Results
* The background should be a **neutral monochromatic color** (light gray or similar).
* Text should be **dark gray or black** for readability.
* Buttons should use **neutral colors** with a clear hover effect.
* The UI should maintain **clear contrast and accessibility**.
